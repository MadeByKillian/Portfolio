# Portfolio Killian Gauchet

Portfolio immersif et haut de gamme conçu pour démontrer mes compétences techniques et mon sens du design.

## 🚀 Aperçu
- **Design :** Futuristic Glassmorphism (Mode sombre profond).
- **Techno :** React + TypeScript + Vite.
- **Animations :** Fluidité cinématique avec Framer Motion.
- **Responsive :** Optimisé pour tous les écrans.

## 🛠️ Stack Technique
- **Framework :** [React 18](https://reactjs.org/)
- **Langage :** [TypeScript](https://www.typescriptlang.org/)
- **Bundler :** [Vite](https://vitejs.dev/)
- **Animations :** [Framer Motion](https://www.framer.com/motion/)
- **Icons :** [Lucide React](https://lucide.dev/)

## 📖 Architecture
Le projet est structuré autour d'un fichier principal `App.tsx` utilisant des styles injectés dynamiquement pour garantir une portabilité maximale tout en conservant une DA haut de gamme.

## 📦 Installation & Déploiement

### Développement
```bash
npm install
npm run dev
```

### Build
```bash
npm run build
```

### Déploiement
```bash
npm run deploy
```
